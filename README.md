# hor1eblePlayer-cli
Плеер для музыки, использует ~Soundcloud~ ~Yandex-music~ Youtube для работы. Будет загрузка треков и многие другие функции.
Полный реворк начат.

## Важно!!!
1.Требуется установленный FFmpeg,а также плеер VLC, помимо библиотек.

2. Если вы проживаете в РФ, требуется впн/запрет

3. Принято решение отложить идею графического интерфейса, в близжайшее время будет релиз терминальной cli-версии

(т.к. это мой выпускной проект, и я не укладываюсь в сроки, а с граф интерфейсом опыт у меня м инимальный) 

## Инструменты для реализации:
1.~pyqt для графического интерфейса(вместо tkinter)~

2.~soundcloud~ ~Yandex-music-api~ youtube-dl для музыки.
Ручная загрузка будет доступна.
