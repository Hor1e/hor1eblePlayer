# hor1eblePlayer-cli Alpha 0.1
## Кратко:
Плеер для музыки, использует Youtube для работы. Проект ориентирован на Linux-системы.


## Убедительная просьба прочитать это пред использованием.

1.Требуется установленный FFmpeg,а также плеер VLC, помимо библиотек.

2. Если вы проживаете в РФ, требуется впн/запрет

3. Это Альфа-версия. Возможно БОЛЬШОЕ количество багов и недочетов, я знаю о них и буду фиксить постепенно.

4. Релиз 1.0, будет когда я доделаю все, что запланировано. 

## Инструменты для реализации:

1. Как-бы это странно ни звучало, но Python.

2. Библиотеки yt-dlp и youtube-search-python для музыки

3. Sqlite3 для манипуляций типа экспорта/импорта плейлиста

## Проект состоит из:

1. Кода на Python

2. 2 shell-скрипта, для запуска из под разных оболочек терминала(fish и другие)

3. Файла с зависимостями


## Быстрый старт
### Ниже представлена инструкция для Arch и его производных
1. Установите FFmpeg(если его нету) с официального сайта.
   
```
sudo pacman -S ffmpeg
```
2. Установите VLC(если его нету) с официального сайта.
   
```
sudo pacman -S vlc
```
3. Установите библиотеки.
```
pip install -r requirements.txt
```
4.Запускайте код.
```
cd 'путь к папке плеера'
bash start.sh
fish start.fish
```
Внимание! Можно получить ошибку ModuleNotFoundError.
Для фикса создайте внутри папки виртуальное окружение Python.
```
python -m venv .venv
source .venv/bin/activate --Это кстати можно дописать в начало shell-скрипта для запуска
```
После этого заново устанавливаете библиотеки, но уже внутри виртуального окружения.
